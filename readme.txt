An easy solution for SSO allowing users logging into WordPress with their Freemius credentials. When embedding the User Dashboard (https://github.com/Freemius/freemius-users-dashboard), a logged in user will be automatically logged into their Freemius User Dashboard without the need to manually log in.

== Description ==


== Installation ==

1. Upload the `freemius-wordpress-sso` folder to the `/wp-content/plugins/` directory
2. Activate the SSO plugin from the 'Plugins' menu in the WordPress Administration section.
3. Open the main plugin's file `freemius-wordpress-sso.php`, scroll to the bottom, and set your store ID, developer ID, and developer secret key.
4. Done!
